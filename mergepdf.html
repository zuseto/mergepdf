Cara Penggunaan:

  • Klik atau seret beberapa file PDF ke kotak di atas.
  • Atur urutan file dengan menyeret pratinjau (desktop), atau gunakan tombol ↑ ↓ dan sentuh-tahan-seret di perangkat mobile/tablet.
  • Klik ikon putar untuk memutar file 90°, atau ikon hapus untuk membuang file dari daftar.
  • Setelah urutan benar, klik Gabungkan PDF untuk memproses dokumen.
  • Tombol Unduh PDF Gabungan akan muncul setelah dokumen selesai diproses.
  • Gunakan tombol Batalkan & Reset untuk menggabungkan PDF lainnya.

* Semua proses berjalan di perangkat Anda (di dalam browser). File PDF tidak diunggah ke server mana pun, jadi tetap privat & aman.
